# GraphQL API for [confs.tech](http://confs.tech)

### Setup

``` bash
yarn install
```

### Runing in dev server

``` bash
yarn start
```

Open your browser to [http://localhost:3000](http://localhost:3000)


### Deployment

``` bash
yarn run deploy
```

### Deployment Logs

```bash
npm install now-logs --g
```

```bash
now-logs confs
```
